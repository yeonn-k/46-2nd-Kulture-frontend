## 🪴 서비스 소개
- 서비스명 : kulture
- 판매상품 : 공연 예술 티켓
- 고객 : 문화생활 및 기부를 통한 사회적 가치 창출에 관심있는 2030
- 컨셉
	- 경매/즉시 구매 두 가지 형태로 공연예술 티켓을 구매할 수 있는 플랫폼
	- 티켓 구매 금액의 10%는 저소득층의 문화생활 지원사업에 자동으로 기부
